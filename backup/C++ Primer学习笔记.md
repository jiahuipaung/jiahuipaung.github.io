# 第六章 函数
## 含有可变形参的函数
1. initializer_list形参，解决形参数量未知，但所有形参类型均相同的问题。其和vector一样是模版类型，不同点是其生命的恶对象中的元素永远是常量值，无法改变。有begin和end操作；可在函数内使用范围for
2. 省略符形参：一般只用于与C函数交互；只能出现在最后一个参数位置；无需进行类型检查；应仅用于C和C++通用的类型

> 注意结尾是否有返回值，该问题编译器可能无法发现
> 不要返回局部变量的指针或引用
> 函数返回引用时得到左值，返回其他时得到右值

## 返回数组指针
1. 使用typedef int arrT[10]或using arrT = int[10]来定义返回类型为数组的指针；arrT为含有10个整数的数组的别名
2. Type  (*func(params_list)) [dim]
3. 尾置返回类型：auto func(int i) -> int(*)[10];
4. 使用decltype：int odd[] = {1,2,3};   decltype(odd) *func(int i);

## 函数重载
1. 可以参数不同，不能只有返回类型不同；
2. 拥有顶层const的形参不能和没有顶层const的形参区分开；但是底层可以（麻烦死了）
